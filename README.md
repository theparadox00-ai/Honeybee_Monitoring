# Honeybee_Monitoring
Ultra-low-power ESP32-based IoT data logger designed for continuous honeybee hive monitoring, Using the 2000mAh LiPo efficiently through DS3231 RTC deep sleep (10ÂµA) and intelligent task scheduling. System integrates HX711 load cell for hive weight tracking (30s intervals), SHT45 temperature/humidity sensors (15min), MAX17043 fuel gauge, and audio detection via polymorphic task scheduler managing I2C/SPI peripherals, SD card FAT32 logging , and Gmail SMTP CSV transmission every 3 hours. Features boot-count state persistence, night-mode power gating.
