# Honeybee_Monitoring
Ultra-low-power ESP32-based IoT data logger designed for continuous honeybee hive monitoring, achieving 4-month battery life on 2000mAh LiPo through DS3231 RTC deep sleep (10ÂµA) and intelligent task scheduling. System integrates HX711 load cell for hive weight tracking (20s intervals), SHT45 temperature/humidity sensors (30min), MAX17043 fuel gauge, and audio detection via C++ polymorphic task scheduler managing I2C/SPI peripherals, SD card FAT32 logging with wear-leveling, and Gmail SMTP CSV transmission every 5 hours.
